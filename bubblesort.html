Bubble Sort

How it works:

Bubble sort is a sorting algorithm that compares the first element of an array with the second element. If the first element is greater than the second, they swap places and this tehnique will be executed untill the array if finished.

This algorithm has a time complexity of O(n2) and a space complexity of O(1).
